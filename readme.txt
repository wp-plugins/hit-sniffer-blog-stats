== Description ==


Hit Sniffer is a powerful real time visitors activity tracker.
Monitor your visitors live and get useful stats data such as referrers, bounce rate,
search keywords, technologies used, OS, Browser, Came from, Country,
Timely Graph and much more... All in Live Realtime Stream.
just open live section and watch your visitors stream.
It works on both Blogs and Websites.


== Installation ==

To use this plugin, you must have a hitsniffer.com account. Get your free account now!


1. Upload `hit-sniffer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Register your website in hitsniffer.com
4. Open setting of your added site in hitsniffer.com and copy/get Wordpress API Code.
5. Open your Wordpress admin Setting, and open hit sniffer sub page, Paste/Put API code.
5. Now, your site is being tracked. you can watch your stats info on hitsniffer.com dashboard and on wordpress dashboard widget.

View Hit Sniffer Features:
http://www.hitsniffer.com/features.php

Cannot get hit sniffer to work on your site?
